# Desafío Alura - Ciencia de Datos

## Descripción del Proyecto

Análisis de datos para recomendar al Sr. Juan qué tienda priorizar en su inversión, evaluando:

* Facturación total
* Ventas por categoría
* Satisfacción de clientes
* Costos logísticos
